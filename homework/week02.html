Abstraction

Explain the meaning of Abstraction

- Abstraction is the proccess of understanding complex idea and dividing it into smaller or simple proccess. At the end making it look easier and more understandable.

Highlight a benefit of Abstraction

- Abstraction help to reduce the complexity of a code

- Makes it easier to read and maintain a code or programe

- It helps programmers to reuse a code or function instead of rewriting a code over and over again.

Application of Abstraction

- programmers use abstraction in Object and Class oriented programming. For instance a building can be seen as an object, that holds so many part of the building, the windows, door bed etc While a class can be seen as a method that carries out the action, example calculate the size of the windows, doors bed etc
