Inheritance

Explain the meaning of Inheritance

- Inheritance in proramming is just the same in real world. It is a concpet where one class inherites, or automatically gain access to methods and attributes from another class.

Highlight a benefit of Inheritance

- Inheritance helps reduces duplication, centralizes common logic.

- Inheritance Makes it easier for programmers to maintain code and avoide too many changes when developing a program.

- It helps programmers to reuse a code or function instead of rewriting a code over and over again.

Application of Inheritance

- Inheritance are mainly use in letting a class obtain attributes and methods of another class that is called the parent class.

- For example. In the Mindfulness program I created, Activity is a base class that holds shared data (name, description, duration) and common helper methods (showing the starting/ending messages, spinner animation, countdown). Each specific activity — breathing, reflecting, and listing — inherits from Activity.

***Base class (shared behavior)***

public class Activity

protected string _name;

protected string _description;

protected int _duration;

public void DisplayEndingMessage()

public void DisplayStartingMessage()

***Derived class (inherited behavior)***

public class ReflectingActivity : Activity

DisplayStartingMessage(); (inherited method)

DisplayEndingMessage(); (inherited method)
